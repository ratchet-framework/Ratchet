Ratchet Memory

Adaptive memory that survives session boundaries — facts extracted automatically from transcripts, scored by importance and decay, retrieved by semantic similarity.

How it works

Three primitives, four lifecycle stages

  • ⚗️
    Extract

    At session end, an isolated LLM agent reads the transcript and pulls discrete, atomic facts — never inferences. "Aaron's WRX oil change is due at 11,799 miles" is a fact. "Aaron seems to care about his car" is not.

  • 📦
    Store

    Facts land in flat facts-YYYY-Q#.jsonl files — append-only, portable, no database required. Each fact carries an importance score, tags, category, and a supersedes field to prevent contradictions from accumulating.

  • 🔍
    Retrieve

    On session start, the top 10–15 relevant facts are injected as "Known context from previous sessions." Phase 1 uses LLM-based selection; Phase 3 switches to cosine similarity embeddings once the fact store exceeds 50 entries.

  • 🔄
    Manage

    Weekly: facts decay at 0.95×/week unreferenced, reinforce when the user confirms them (injection doesn't count — prevents runaway loops), promote to MEMORY.md when importance > 0.8 and reference count ≥ 3, and purge when effective score drops below 0.1.

  • 🛡️
    Prompt injection defense

    The extraction agent has no tool access. Content from untrusted sources (email body, web pages) is wrapped in [UNTRUSTED DATA] delimiters. Facts from untrusted sources get a 0.5× importance penalty and require a higher threshold for promotion.

See it in action

Scoring and lifecycle

Score = base_importance × recency_boost × relevance_to_context

Three separate axes — not collapsed into one number until retrieval time. This means a highly important but stale fact can still be surfaced when relevant, and a recent but trivial fact doesn't crowd out critical context.

The Opus critique that shaped this design: "The core tension: you want the extractor to be creative enough to identify implicit facts but constrained enough to never hallucinate. Err hard toward literal extraction. A missed fact is recoverable — it'll surface again. A hallucinated fact poisons the memory store and compounds over time."

Fact structure on disk:

{
  "id": "uuid",
  "content": "WRX oil change due at 11,799 miles",
  "category": "vehicle",
  "importance": 0.85,
  "tier": "standard",       // permanent | standard | transient
  "created": "2026-02-28",
  "last_referenced": "2026-03-01",
  "reference_count": 2,
  "supersedes": null,
  "source_trust": "trusted"
}
